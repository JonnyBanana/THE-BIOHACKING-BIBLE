{{SfGS}} Hi gang, Thanks for helping to develop our SfGS wiki! Here are a few quick thoughts to help orient you to our new wiki space: * The wiki is essentially a set of linked pages that everyone can edit for ease of information sharing. * Each of our SfGS pages has a name of the format: ''SfGS: page_name'' * To edit a page, click on the "edit" tab at the top of the page of interest. This will take you to a text box where you can edit the page. * To link to an '''existing''' page (e.g. the SfGS hompage), just add the following text: [[SfGS: Homepage]] * To make a '''new''' page, simply make a link to your new page in an existing page. Page links look like this: [[SfGS: page_name_here]] *If you want to link to a page (e.g. SfGS: Publications) but have the link name look different than the page name (e.g. Stuff we made), you can write the following: [[SfGS: Pubilcations| Stuff we made]] *To add an SfGS navigator menu in the upper right hand of a page, be sure to type {{SfGS}} at the top of your new page. * To make a word '''bold''', add three apostrophes on each side of the word, as shown here: '''something''' * To make something ''italic'', add two apostrophes on each side of the word, as shown here: ''something'' * To add an html link, use the following format: [http://your.website.org Link_name] * To add an file (e.g. Schedule.doc) to a page (e.g. SfGS: Foo), do the following: ** Click on the "upload file" link in the left hand toolbar ** Follow instructions.
